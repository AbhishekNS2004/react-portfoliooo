- **Dynamic Project Showcase**: A custom-built carousel showcasing my latest full-stack and AI/ML projects.
- **Responsive Design**: Fully optimized for all screen sizes with a "premium" dark-mode aesthetic.
- **Performance Optimized**: Optimized assets and lazy loading for a lightning-fast experience.

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AbhishekNS2004/react-portfoliooo.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd abhi-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
